### KeenKeeper is a simple app to help you remember to talk to your friends. It keeps a list of your favorite people and shows you exactly how many days it has been since you last messaged or called them. 

-----
### Key Features :
* 1. See all your friends in one place if you haven't talked to them in a while.
* 2. A clear list that shows the date and type (Call, Text, or Video) of every time you reached out.
* 3. A view profile option to see all the friend details
* 4. A colorful pie chart that shows if you prefer calling, texting, or video chatting.

-------
